=====================================
 redirectstore convenience functions
=====================================

opencore.redirect provides a basic convenience functions for
managing redirectstore mappings::

    >>> from opencore.redirect.site import add_redirectstore
    >>> add_redirectstore(self.folder)
    >>> self.folder.utilities.objectIds()
    ['IRedirectMapping']

    >>> from opencore.redirect.site import get_redirectstore
    >>> get_redirectstore(self.folder)
    <OFSRedirectStore at /test_folder_1_/utilities/IRedirectMapping>

    >>> from opencore.redirect.site import del_redirectstore
    >>> del_redirectstore(self.folder)
    >>> self.folder.utilities.objectIds()
    []
